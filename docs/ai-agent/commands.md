# Quick Commands Reference

```bash
# Development
npm run dev
npm run storybook

# Testing
npm run test
npm run test:watch
npm run test:coverage
npm run test:e2e
npm run test:e2e:ui
npm run test:storybook
npm run test:a11y
npm run test:mutation
npm run test:all

# Type & Quality
npm run type-check
npm run lint
npm run lint:fix
npm run format
npm run format:check

# Validation
npm run validate
npm run validate:all
npm run validate:i18n

# Build
npm run build
npm run preview
npm run build-storybook
```

**Before commit:** `npm run validate:all`

**Slash commands (Claude/Cursor):** `/verify`, `/docs-sync`, `/pr-create`, `/pr-fix`, `/issue-implement <n>`, `/issue-refine <n|text>`, `/handoff`
