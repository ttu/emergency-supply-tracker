# Emergency Supply Tracker

Web-based emergency supply tracking application (based on 72tuntia.fi guidelines).

## Tech Stack
- React 19 + TypeScript 5.9 + Vite 7
- LocalStorage (no backend)
- GitHub Pages or Render.com deployment

## Development
```bash
npm install
npm run dev
```
